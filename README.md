# SpringBootApp

A simple Spring Boot REST API for managing students, courses, and grades using Spring Web MVC, Spring Data JPA, and an in-memory H2 database.

## Setup

1. Install Java 17 and Maven.
2. Run the app:
   ```bash
   mvn spring-boot:run
   ```
3. API base endpoints:
   - `/students`
   - `/courses`
   - `/grades`
